# B7A1 : Advanced Problem Solving with TypeScript & OOP

This repository is my submission for the B7A1 assignment. It includes 7 TypeScript problem solutions covering arrays, strings, union types, generics, interfaces, OOP, and type guards along with two blog posts on TypeScript concepts.

## File Structure

```
├── solutions.ts
├── blog-1.md
├── blog-2.md
└── README.md
```

## Solutions (`solutions.ts`)

7 problems covering `filterEvenNumbers`, `reverseString`, `checkType`, `getProperty`, `toggleReadStatus`, `Person`/`Student` class inheritance, and `getIntersection`.

## Blog Posts

- **blog-1.md** : Why `any` is unsafe and how `unknown` with type narrowing is the better choice.
- **blog-2.md** : How Generics let you write reusable functions and components that stay strictly typed.
